# vue-ssr-starter-kit

> A Vue.js project with vue 2.0, vue-router and vuex starter kit for server side rendering.

## Node Version Requirement

```bash
node 6.*
node 7.*
```

## Development Setup

```bash
npm install
npm run dev
```

## Build Setup

```bash
npm install
npm run build
npm start
```
