## Avatar

用来代表用户或事物，支持图片、图标或字符展示。

## 使用

```html
<m-avatar size="60" src="avatar.jpg"></m-avatar>

<m-avatar icon="person"></m-avatar>
```


## API

### Props

```jsx
{
    alt?: string;
    src?: string;
    size?: number;
    square?: boolean;
    icon?: string;
    color?: string;
    backgroundColor?: string;
}
```
