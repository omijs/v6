## Date Range Picker

Date Range Picker.

## Usage

```html
<m-date-range-picker
  left-date="2018-6"
  right-date="2019-10"
  from="2018-6-9"
  to="2018-10-10"
>
</m-date-range-picker>
```

## API

### Props

```jsx
{
  from?: string,
  to?: string,
  leftDate?: string,
  rightDate?: string
}
```
