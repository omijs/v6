## Row

24 Grids System.

## Usage

```html
<m-row cols="[8, 8, 8]">
  <div>col-8-0</div>
  <div>col-8-1</div>
  <div>col-8-2</div>
</m-row>
```

## Usage in Omi

JSX:

```jsx
<m-row cols={[8, 8, 8]}>
  <div>col-8-0</div>
  <div>col-8-1</div>
  <div>col-8-2</div>
</m-row>
```

## API

### Props

