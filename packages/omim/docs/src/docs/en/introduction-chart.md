## Install

### Via script

```html
<script src="https://unpkg.com/omi"></script>
<script src="https://unpkg.com/omim@latest/chart/index.js"></script>
```

### Via npm

``` bash
npm install omim
```

Then:

```js
import 'omim/chart'
```
