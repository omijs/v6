## Avatar

Used to represent users or things, supporting the display of pictures, icons or characters.

## Usage

```html
<m-avatar size="60" src="avatar.jpg"></m-avatar>

<m-avatar icon="person"></m-avatar>
```


## API

### Props

```jsx
{
    alt?: string;
    src?: string;
    size?: number;
    square?: boolean;
    icon?: string;
    color?: string;
    backgroundColor?: string;
}
```
